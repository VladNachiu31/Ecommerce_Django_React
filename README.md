# Ecommerce_Django_React

This is a full-stack e-commerce website with the following features: <br>
1 - Custom Django admin interface where you can add and receive orders <br>
2 - The ability to Register/Login <br>
3 - Payent process using Paypall <br>
4 - Friendly UI <br>

To get started with the project <br>
BACK END - Install the requirements.txt with the command pip install -r requirements.txt in your shell. <br>
Navigate into the backend folder: <br>
1 - type paython3 manage.py run server <br>

FRONT END - Navigate in the terminal into frontend folder: <br>
1 - type npm install <br>
2 - type npm start <br>
